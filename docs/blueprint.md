# **App Name**: AirWave AC Manager

## Core Features:

- AC Unit List: Display a filterable list of AC units, with filtering by company, city, and status.
- AC Unit Add/Edit: Provide a detailed form to input AC unit data including model, serial, inverter status, brand, BTU, gas type, AC type, ownership, location, city, status, and map location with customer fields appearing when relevant
- Data Export: Offer a one-click export of all AC unit data to a file.
- Data Import: Offer a one-click import of AC unit data from a file.
- Configuration Management: Allow configuration of AC-related constants, with simple controls to add companies, gas types, AC brands, and BTU capacities.

## Style Guidelines:

- Primary color: A calming light blue (#75A8D9) reminiscent of cool air, suggesting efficiency and reliability in AC management.
- Background color: A very light blue-gray (#F0F4F7), maintaining a clean and professional look.
- Accent color: A gentle, muted purple (#9275D9) for interactive elements and highlights, chosen as an analogous color to the primary with a slight variation to add visual interest.
- Font: 'Inter', a sans-serif font that will provide a modern and clean aesthetic. It is suitable for both headlines and body text. Note: currently only Google Fonts are supported.
- The app layout should utilize clear, concise form elements to provide an optimal user experience when creating new or modifying existing AC units.
- Implement subtle transitions when displaying and filtering content in the system. This will provide a better user experience and indicate processing
